# API for An's Home project.

## Technologies

- Nodejs(express)
- MongoDB

## Installation

- clone project: https://github.com/dinkloc/an-home
- cd at root project, install all dependencies
  > npm install
- run project
  > npm start

## Feature

- Authentication, Authorization, Forgot Password
- Crud Bookings, Cabins,......
- Pagination, statistics,....
